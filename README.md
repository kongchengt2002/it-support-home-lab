# IT Support Home Lab

## Overview

This IT Support Home Lab project was created to practice beginner help desk and IT support skills. The project includes troubleshooting documentation, Windows command-line practice, simulated support tickets, and screenshots showing common IT support tasks.

The goal of this project is to build hands-on experience with technical troubleshooting, documentation, and user support scenarios that are commonly used in entry-level IT support, help desk, and desktop support roles.

## Project Goals

* Practice common IT troubleshooting steps
* Document support issues like a real help desk technician
* Use Windows tools such as Command Prompt and system settings
* Create sample support tickets with problems, solutions, and resolutions
* Build a project that demonstrates technical learning and documentation skills

## Project Sections

### 1. Troubleshooting Notes

This section includes notes on common IT support problems such as slow computers, network connection issues, login problems, software errors, and basic Windows issues. Each issue includes possible causes and troubleshooting steps.

### 2. Command Prompt Practice

This section documents basic Windows command-line tools used in IT support, including:

* `ipconfig`
* `ping`
* `tracert`
* `nslookup`
* `systeminfo`

These commands are useful for checking network settings, testing connectivity, and gathering system information.

### 3. Simulated Support Tickets

This project includes sample support tickets written like real help desk cases. Each ticket includes:

* Issue description
* User problem
* Troubleshooting steps
* Resolution
* Final status

Example ticket topics include:

* User cannot connect to Wi-Fi
* Computer is running slowly
* Password reset request
* Printer not responding
* Application will not open
* Email login issue

### 4. Screenshots

Screenshots are included to show proof of hands-on practice. These screenshots may include:

* Command Prompt commands
* Windows settings
* Network troubleshooting
* Ticket documentation
* File organization

Sensitive information such as IP addresses, usernames, device names, Wi-Fi names, and personal information should be blurred or removed before uploading.

## Project Contents

- **1 - Ticket Log**  
  Contains a spreadsheet of simulated help desk tickets with issue categories, priority levels, troubleshooting steps, resolutions, and final status.

- **2 - Troubleshooting Guides**  
  Includes step-by-step troubleshooting guides for common Tier 1 support issues such as Wi-Fi problems, slow computers, software installation issues, Microsoft Teams audio problems, and browser cache issues.

- **3 - Screenshots**  
  Shows proof of hands-on practice with Windows settings, Command Prompt tools, networking checks, printer settings, sound settings, and support documentation.

- **4 - Commands Practice**  
  Includes documentation of basic Windows support commands such as `ipconfig`, `ping`, `nslookup`, `tracert`, `systeminfo`, `tasklist`, and `sfc /scannow`.

- **5 - Resume and LinkedIn Proof**  
  Contains a project summary that can be used for LinkedIn, resumes, and job applications.
